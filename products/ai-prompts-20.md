## はじめに

このプロンプト集は、日々のビジネスシーンで即使える実践的なプロンプトを厳選しました。
ChatGPT、Claude、Geminiなど、どのAIでも使えます。

**使い方のコツ：**
- [ ]内の部分を自分の状況に置き換えて使う
- 出力が不十分なら「もっと詳しく」「別の案も」と追加で指示する
- 良かったプロンプトはカスタマイズして自分専用に育てる

## 📧 ビジネスメール（5選）

### 1. 丁寧な断りメール

```
以下の依頼を丁寧に断るメールを書いてください。
相手との関係性を損なわず、今後の可能性は残しつつ、
明確に今回はお断りする内容にしてください。

【断る依頼の内容】
[例：来週の会議への参加依頼]

【断る理由】
[例：別件の締め切りが重なっている]

【相手との関係性】
[例：取引先の担当者、3年の付き合い]
```

### 2. 催促メール（角が立たない）

```
以下の件について、角が立たないように催促するメールを書いてください。
相手を責めるニュアンスは避け、こちらの状況を伝えつつ、
対応をお願いする内容にしてください。

【催促したい内容】
[例：見積書の送付]

【最初に依頼した日】
[例：1週間前]

【急ぎ度合い】
[例：今週中に必要]
```

### 3. お詫びメール

```
以下の件についてお詫びのメールを書いてください。
言い訳がましくならず、誠意が伝わる内容にしてください。
今後の対策も含めてください。

【お詫びする内容】
[例：納品が2日遅れた]

【原因（書ける範囲で）】
[例：社内の確認プロセスに時間がかかった]

【今後の対策】
[例：チェック体制を見直す]
```

### 4. 初めての相手へのアポイントメール

```
面識のない相手にアポイントを取るメールを書いてください。
押し付けがましくなく、相手にメリットが伝わる内容にしてください。

【自分の立場】
[例：株式会社〇〇の営業担当]

【相手の立場】
[例：△△株式会社のマーケティング部長]

【アポの目的】
[例：新サービスのご紹介]

【相手にとってのメリット】
[例：業務効率化につながる可能性がある]
```

### 5. 感謝メール（印象に残る）

```
以下の件について感謝を伝えるメールを書いてください。
形式的にならず、具体的に何が助かったかを伝え、
印象に残る内容にしてください。

【感謝したい内容】
[例：急な依頼に対応してくれた]

【具体的に助かったポイント】
[例：おかげでクライアントへの提出が間に合った]
```

## 🎯 活用のコツ

### 効果を最大化する3つのポイント

1. **具体的に書く**
   - ❌「メールを書いて」
   - ⭕「取引先への謝罪メールを書いて。納期が3日遅れた件。」

2. **出力形式を指定する**
   - 「箇条書きで」「表形式で」「〇〇字以内で」

3. **フィードバックする**
   - 「もっとカジュアルに」「専門用語を減らして」「具体例を追加して」

### カスタマイズのすすめ

このプロンプト集をそのまま使うのも良いですが、
自分の業界・職種に合わせてカスタマイズすると、さらに効果的です。

例：
- 業界特有の用語を追加
- よく使うフォーマットを組み込む
- 自社のトーン&マナーを指定
